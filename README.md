# Superstore
This project consists of a front-end written in React.js, a back-end written in C#, and MS SQL Stored Procedures.

## Superstore API

### Product
- **GET** /api/products
- **GET** /api/products/{id}
- **POST** /api/products
- **PUT** /api/products/{id}
- **DELETE** /api/products/{id}

### AddressType (Lookup Table)
- **GET** /api/addresstypes
